About Me

I'm Vaishnavi, a passionate Electronics & Communication Engineering student who loves working on robotics and web development projects. With hands-on experience in building intelligent systems and creating interactive web applications, I aim to bridge hardware with software in meaningful ways.

I've built projects ranging from face recognition-based robots to QR code generators, and I'm always excited to learn and take on new challenges.

Currently I am working on an Office Assistant Robot. This project integrates hardware and software to automate faculty identification, presence tracking, and academic query resolution in an academic environment.

Purpose & Impact:

This robot aims to streamline administrative operations in educational institutions by automating repetitive tasks and assisting faculty members with real-time information. It demonstrates the real-world application of robotics, machine learning, and web development in a cohesive system.

Skills: Python, HTML, CSS, JavaScript, Flask, OpenCV, SQL, Arduino, Raspberry Pi
